# Software Developer Portfolio

A modern, responsive portfolio website built with React. This portfolio showcases your work, skills, education, and experience in a beautiful landing page format.

## Features

- **Home**: Welcome section with hero content
- **Summary**: About me section with technical skills
- **Portfolio**: Showcase of projects and work
- **Education**: Educational background and certifications
- **Experience**: Work experience timeline

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Customization

You can customize the portfolio by editing the component files:

- Update personal information in each component
- Modify the color scheme in CSS files
- Add your own projects, education, and experience
- Adjust styling to match your preferences

## Technologies Used

- React 18
- Vite
